# PlatinumRx Data Analyst Assignment

This repository contains my solutions for the NxtWave PlatinumRx Data Analyst assignment.

## Project Structure
- **SQL/**: Contains SQL scripts for Hotel and Clinic Management database schemas and queries.
- **Spreadsheets/**: Contains a text file with the link to the Google Sheet used for Ticket and Feedback analysis.
- **Python/**: Contains Python scripts for time conversion and string manipulation.

## How to Check My Work
1. **SQL**: The queries can be executed in any standard SQL environment (like SQLite or MySQL).
2. **Spreadsheet**: The provided Google Sheets link is public. It demonstrates the use of VLOOKUP and COUNTIFS.
3. **Python**: The scripts are written in Python 3.x and can be run in any IDE or online compiler.
